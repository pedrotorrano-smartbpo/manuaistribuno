Instruções

A tela inicial é o primeiro contato com a plataforma que o usuário tem após fazer o login. Nessa tela temos algumas informações e componentes que serão descritos abaixo:

  1. Menu lateral: este menu pode variar de acordo com as permissões de cada usuário.
  2. Analítico (Dashboards): gráficos com informações relacionadas as proposições do usuário.
  3. Seleção de visualização: permite ao usuário selecionar diferentes tipos de visualização do gráfico.
  4. Temas monitorados: filtro dos temas monitorados, permite escolher quais deseja visualizar.
  5. Menu do usuário: menus relacionados ao usuário como mudar senha e sair do sistema.
  6. Proposições: área que demonstra as proposições relacionadas aos temas do usuário.
