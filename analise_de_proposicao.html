Análise de proposição

Video

Instruções

    1.   Para fazer a análise de uma proposição, clique no ícone azul localizado no final da linha da proposição.
    2.   Uma nova janela com algumas opções editáveis aparecerá.
    3.   As seguintes opções estão disponíveis para analisar a proposição:
    1.   Análise: descrever os possíveis impactos e consequências desse projeto.
    2.   Impacto: qual será o impacto desse projeto para a instituição.
    3.   Resumo: escrever um resumo do projeto de lei.
    4.   Posição: qual a posição da instituição em relação a esse projeto.
    5.   Monitorar: opção para acompanhar qualquer mudança nessa proposição.
    4.   Após realizar a análise, clique no botão “Salvar” para manter os registros.
    5.   As tramitações relacionadas com essa proposição são mostradas abaixo do botão de “Salvar” e “Cancelar”.
